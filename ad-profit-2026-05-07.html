광고 수익분석 통합 대시보드

Google · Naver · Meta를 한 화면에서 관리

광고 전환 주문 기준입니다. Meta는 Supabase 집계값을 그대로 보고, Google Ads와 Naver SearchAd는 채널 광고비/클릭/노출/API 전환값만 raw API 원본으로 다시 맞춥니다. 주문 연결 매출·순이익은 세 채널 모두 Supabase 집계값을 유지합니다.

채널별 수익분석

선택 기간의 광고 채널별 지출과 주문 연결 순이익을 비교합니다. 채널 요약은 `vw_marketing_profit_query_daily`를 읽고, Google Ads와 Naver SearchAd의 광고 API 수치만 raw snapshot으로 덮어씁니다.

채널별 기간 합계

선택 기간에 합산된 채널별 광고비를 바로 확인합니다. Meta는 DB 기준, Google Ads는 `metrics.cost_micros / 1,000,000`, Naver SearchAd는 `salesAmt` 기준입니다.

채널 집계일수 광고비 합계 광고매출(API) 주문 연결 매출 주문 연결 순이익

채널별 일별

Meta, Naver SearchAd, Google Ads를 각각 분리해서 최근 일별 광고비와 주문 연결 실적을 봅니다. Google Ads와 Naver SearchAd는 광고비/광고지표를 raw API 일별 원본으로 표시합니다.

채널별 월별

Meta, Naver SearchAd, Google Ads의 월 누적 광고비와 주문 연결 실적을 채널별로 분리해 봅니다. Google Ads와 Naver SearchAd는 월 누적 광고비를 raw API 합계로 계산합니다.

자사몰 + 네이버 카테고리별 운영 순익

Supabase 주문/원가 기준으로 자사몰과 네이버 주문을 합산합니다. 계산식은 매출 - 원가 - 채널수수료 - 배송비이며, 광고비는 이 섹션에서 제외합니다.
